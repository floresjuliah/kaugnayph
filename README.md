# KaugnayPH

## Overview
KaugnayPH is a web-based barangay management and communication system developed for Barangay 761, Zone 82, District V, Manila. The system streamlines barangay services by providing online document requests, complaint management, announcements, inquiries, resident account verification, SMS notifications, and administrative tools for efficient barangay operations.
